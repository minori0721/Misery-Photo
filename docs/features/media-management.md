# 媒体管理

## 浏览模式

系统提供两种主要浏览模式：

- 网格模式：适合快速筛选与管理。
- 漫画模式：适合顺序浏览连续内容。

## 支持的媒体类型

- 图片：jpg、jpeg、png、webp、gif
- 视频：mp4、webm、mov、m4v、m3u8

## 视频能力

- 列表中可识别视频文件并展示播放入口。
- 支持在弹层中播放视频。
- 对 m3u8 资源可通过 hls.js 兼容播放。

## 预览与性能

- 文件夹预览优先使用图片，避免视频首帧影响加载体验。
- 代理接口支持对大图进行压缩输出，降低首屏流量开销。

## 使用建议

- 大体积视频建议使用外部 CDN 加速。
- 对长期归档内容，建议按画集目录分层避免单目录对象过多。
