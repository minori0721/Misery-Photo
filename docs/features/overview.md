# 功能概览

Misery-photo 的功能围绕三个方向设计：媒体管理、操作效率、安全与可维护性。

## 1. 媒体管理

- 图片与视频混合展示。
- 网格与漫画模式浏览。
- 视频在线播放（含 HLS 资源支持）。

详见 [媒体管理](/features/media-management)。

## 2. 上传与下载

- 单文件上传：图片与视频。
- ZIP 批量导入：自动提取媒体文件。
- 批量下载与打包导出。

## 3. 批量操作

- 批量删除。
- 批量复制与移动。
- 目标目录选择器。

详见 [批量操作](/features/batch-operations)。

## 4. 存储与安全

- 多桶配置持久化。
- 默认桶兜底与切换。
- Access Key / Secret Key 服务端加密存储。

详见 [桶与安全](/features/buckets-and-security)。

## 5. 运行模式说明

- 浏览列表优先使用签名模式（前端解析列表 XML）。
- 上传采用服务端签名 + 浏览器直传。
- 大图代理可开启压缩缩略图模式，降低带宽与首屏等待。
