# 桶与安全

## 多桶管理

- 可保存多个桶配置并切换当前激活桶。
- 可设置默认兜底桶（来自环境变量）。
- 桶状态可持久化在 Upstash/Cloudflare KV。

## 密钥安全

- 桶 Access Key / Secret Key 在服务端加密存储。
- 前端不会持久化明文密钥。
- 读取桶配置时默认返回脱敏视图。

## 会话与鉴权

- 管理操作依赖登录会话。
- 登录接口带失败延迟与基础限流。
- 登出时会清理会话与旧桶状态 Cookie。

## 代理安全

- 仅允许 HTTPS 目标地址。
- 仅允许白名单域名。
- 上游请求有超时控制，避免资源长期占用。

## 生产建议

- 定期轮换管理员密码与密钥。
- 只给 KV Token 最小必要权限。
- 对公网部署启用可信来源限制与访问监控。
