# 介绍

Misery-photo 是一个基于 Next.js 的私有媒体管理系统，面向 S3 兼容存储场景，适用于个人与小团队的图片/视频管理需求。

## 解决的问题

- 在对象存储中统一管理图片与视频资源。
- 在不自建复杂后台的前提下完成上传、浏览、下载和批量处理。
- 通过服务端鉴权与密钥加密降低配置泄露风险。

## 核心能力

- 多存储桶配置与切换。
- 图片与视频统一列表浏览。
- ZIP 媒体上传与批量操作。
- 服务端鉴权与密钥加密存储。

## 适用人群

- 想用 S3/R2/OSS 搭建私有图床或素材库的个人开发者。
- 需要按画集管理图片和视频内容的小团队。
- 希望在 Vercel 快速部署、并保留自建迁移能力的项目维护者。
