# 常见问题

## 1. 为什么登录后看不到桶配置？

优先检查 KV 连接与环境变量是否完整，尤其是 BUCKET_STORE_PROVIDER 与对应 Token。

## 2. 为什么部署后历史桶解密失败？

通常是 BUCKET_ENCRYPTION_KEY 变更导致，旧密文无法被新密钥解密。

## 3. 项目是否只能部署到 Vercel？

不是。只要运行环境支持 Next.js 服务端并可访问外部 KV，也可以自建部署。

## 4. 文档站在 Vercel 构建失败，提示 PostCSS/Tailwind 找不到模块

请确认：

1. 文档项目 Root Directory 为 docs。
2. docs 目录存在独立 PostCSS 配置文件。
3. 文档项目 Framework Preset 为 VitePress。

## 5. 为什么会出现 401 未授权

常见原因：

- 登录会话已失效。
- AUTH_SECRET 未配置或变更。
- 使用了错误的部署环境变量集。

## 6. 上传成功但列表看不到文件

建议检查：

1. 当前激活桶是否正确。
2. 上传路径是否与当前目录一致。
3. 对象存储权限和 endpoint 是否正确。

## 7. 批量移动特别慢怎么办

- 减小单次路径数量。
- 按目录分批执行。
- 优先在同一区域桶内操作，减少跨区域延迟。
